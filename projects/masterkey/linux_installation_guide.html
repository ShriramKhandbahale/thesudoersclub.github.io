MasterKey installation guide

  1. Step 1: Change the directory to where the file is downloaded.
    2. After downloading the file it will be saved in a directory on you computer. It should be the downloads directory for most of the people, change the path after cd to the location where the file is downloaded on your computer
    $ cd /home/$USER/Downloads/
  2. Step 2: Unzip the downloaded file
    3. After changing the directory to where the file is downloaded use the unzip command to extract the downloaded folder in the same directory.
    $ unzip masterkey.zip
  3. Step 3: Run masterkey_launcher.out
    4. After extracting the folder run the program 'masterkey_launcher.out' to setup the app environment and you should be good to go.
    $ ./masterkey_launcher.out
